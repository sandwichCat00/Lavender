# Lavender

A silly little language I made for fun. Lavender is bytecode-compiled. This repository contains the source code for its compiler and interpreter.

*This is an unfinished iteration and still needs many bug fixes and improvements.*

---

## Goals

- Fix known memory leaks  
- Complete error reporting  
- Add support for arrays and strings  
- Create a basic standard library  
- Add support for structs and namespaces  

---

## Project compilation

### Requirements
- Zig >= 0.15.2

Clone the repository and build:

```bash
git clone https://github.com/sandwichCat00/Lavender.git
cd Lavender
zig build
```
