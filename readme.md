# Lavender

A silly little language I made for fun. Lavender is bytecode-compiled. This Repo contains source code for its compiler/interpreter.

**(this is an unfinished iteration and needs many bug fixes and improvements)**


---

## Goals

- Fix known memory leaks
- Complete error reporting
- Add support for arrays/strings
- Create a basic standard library
- Add support for structs/namespaces
---

## Project compilation

Requirements:

- zig v0.15.2+


Clone the repository:

```bash
git clone https://github.com/sandwichCat00/Lavender.git
cd Lavender
zig build 
```
